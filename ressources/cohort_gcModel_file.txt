THIS IS WHERE THE USER SHOULD PUT THE GC MODEL FILE

This is a tab delimited file with the header below:

Name	Chr	Position	GC

header column definition:
Name: The array genotyping markers SNP name
Chr: SNP chromosome
Position: SNP genomic position as referenced by its chromosome
PFB: GC reach percent, computing by the following #script require# (PFB data is require for this step)
